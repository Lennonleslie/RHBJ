# 🍶 醉仙居 · 中式酒馆美酒鉴赏

一个专为展示白酒（或各类酒品）设计的轻量级交互页面，适合用作产品目录、品牌展示或教学案例。它采用中式酒馆视觉风格，支持多分类筛选、卡片展示、灯箱品鉴（含画廊、规格、获奖、品鉴建议、自定义二维码）等功能。

---

## ✨ 功能特性

- **中式视觉风格**：古风酒馆配色、书法字体、装饰元素，营造沉浸式品鉴氛围。
- **卡片网格展示**：每款酒以“酒牌”形式呈现，包含酒名、产地、简短描述、香型标签、属性标签和年份。
- **多分类筛选**：支持“全部 / 酱香 / 浓香 / 清香”筛选，一款酒可同时归属多个香型（例如兼香型白酒），在多个分类下均能被找到。
- **灯箱详情品鉴**：
  - 主图 + 画廊缩略图（支持多图切换）
  - 详细描述、规格信息（酒精度、净含量、原料、香型、执行标准）
  - 获奖荣誉、饮用/品鉴建议
  - 自定义区域（左图右文，适合放置微信二维码或品牌标识）
- **交互操作**：
  - 点击卡片进入灯箱
  - 点击主图左/右半区切换画廊图片
  - 点击画廊缩略图直接跳转
  - 触摸滑动主图切换画廊图片（移动端）
  - 键盘 `←` / `→` 切换酒款，`Esc` 关闭灯箱
- **完全数据驱动**：所有内容（酒品数据、图片路径、文本）均通过 `data.js` 配置，无需修改 HTML 或 JS 核心逻辑。
- **响应式设计**：适配桌面、平板、手机等多种屏幕尺寸。

## 📝 数据配置说明（`data.js`）

`data.js` 是项目的核心配置文件，每款酒是一个对象，包含以下字段（**带 ★ 为必填**）：

| 字段 | 类型 | 说明 | 示例 |
| :--- | :--- | :--- | :--- |
| **name** ★ | `string` | 酒名 | `'飞天茅台'` |
| **sub** | `string` | 副标题/后缀，可为空 | `'· 陈年'` 或 `''` |
| **origin** ★ | `string` | 产地 | `'贵州茅台镇'` |
| **description** ★ | `string` | 卡片上显示的简短描述（约30字） | `'酱香典范，空杯留香...'` |
| **types** ★ | `array` | 香型数组，用于底部菜单筛选，可多选（如 `['酱香','浓香']`） | `['酱香']` |
| **tags** | `array` | 属性标签数组，仅卡片底部展示，不参与筛选 | `['陈年','纯粮','限量']` |
| **year** | `string` | 年份（显示为 `📅 2015`） | `'2015'` |
| **image** ★ | `string` | 卡片缩略图路径（建议 500×350） | `'./images/maotai_thumb.jpg'` |
| **largeImage** ★ | `string` | 灯箱主图路径（建议 1200×800） | `'./images/maotai_large.jpg'` |
| **detail** | `string` | 灯箱中详细描述（长文本） | `'飞天茅台酒是...'` |
| **gallery** | `array` | 灯箱中额外图片（画廊），自动去重 | `['./images/maotai_01.jpg', ...]` |
| **specs** | `object` | 规格信息（键值对），灯箱中自动渲染 | `{ alcohol:'53%vol', volume:'500ml', ingredients:'高粱、小麦、水', type:'酱香型', standard:'GB/T 18356' }` |
| **awards** | `string` | 获奖/荣誉信息 | `'1915年巴拿马金奖；中国名酒'` |
| **serving** | `string` | 品鉴/饮用建议 | `'适宜温度：20~25℃...'` |
| **custom** | `object` | 自定义区域（左图右文，适合放二维码） | `{ image:'./images/qrcode.png', text:'扫码关注...' }` |

> **提示**：`types` 数组让一款酒可以同时归属多个香型，例如 `['酱香','浓香']` 的兼香型酒，在点击“酱香”或“浓香”分类时都会被筛选出来。

## 🚀 如何运行

### 方式一：本地直接预览（推荐初学者）

1. 下载或克隆本项目到本地文件夹。
2. 准备您的图片，按照 `data.js` 中配置的路径放入 `images/` 目录。
3. **双击 `index.html` 文件**，浏览器会自动打开（地址栏为 `file://` 协议）。
4. 此时所有功能应正常工作（图片加载可能受 CORS 影响，详见下方“常见问题”）。

> 注意：如果图片无法加载，请改用 **方式二** 或参照“常见问题”处理。

### 方式二：使用本地 HTTP 服务器（推荐开发者）

为避免 CORS 问题并保留未来扩展性，建议使用以下任一方式启动服务：

- **VS Code 用户**：安装 “Live Server” 插件，右键 `index.html` → “Open with Live Server”。
- **Python 3 用户**：在项目目录下执行 `python -m http.server 8080`，然后访问 `http://localhost:8080`。
- **Node.js 用户**：安装 `http-server`（`npm install -g http-server`），然后在项目目录执行 `http-server`。

## 🔧 自定义与扩展

### 添加新酒品

1. 打开 `data.js`，复制现有某个对象（如飞天茅台）的结构。
2. 修改所有字段（名称、产地、描述、图片路径等）。
3. 确保 `types` 数组填写正确的香型（可多选）。
4. 保存文件，刷新页面即可看到新卡片。

### 增加新的筛选维度（如“产地”或“年份”）

1. 在 `data.js` 中为每款酒增加新字段，如 `regions: ['贵州','四川']`。
2. 在底部菜单栏（`index.html`）添加新按钮。
3. 在 `main.js` 的 `setFilter` 函数中增加对应筛选逻辑（参考现有香型筛选）。
4. 在 `renderCards` 中增加过滤条件，使用 `includes` 匹配数组。

### 更换视觉风格

- **字体**：修改 `style.css` 中 `@import` 的 Google Fonts 链接，或替换为系统字体。
- **主色调**：调整 `:root` 中的 CSS 变量（如 `--gold`, `--red-dark` 等）。
- **装饰元素**：修改 `.page-header` 和 `.header-curtain` 中的 Unicode 符号或样式。

## ❓ 常见问题

### 1. 图片无法加载，控制台报 CORS 错误

**原因**：您直接双击 HTML 使用 `file://` 协议打开，而代码中设置了 `img.crossOrigin = 'anonymous'`，导致浏览器因安全策略拦截本地图片。

**解决方案**（任选其一）：
- **方案 A（推荐）**：使用 **Live Server** 或 `python -m http.server` 以 `http://` 协议访问，此时 `crossOrigin` 设置正常工作。
- **方案 B（快速修复）**：在 `main.js` 中找到 `loadLightboxImage` 函数，**删除或注释掉** `img.crossOrigin = 'anonymous';` 这一行。修改后直接双击 HTML 即可加载本地图片（但会损失未来 Canvas 扩展能力）。

### 2. 灯箱中只显示缩略图，不显示大图

**原因**：`data.js` 中某款酒的 `largeImage` 路径配置错误，或 `gallery` 数组顺序导致第一张图不是大图。

**排查**：
- 打开浏览器控制台（F12），查看 `📸 [酒名] 加载图片 #1: ...` 日志，确认加载的 URL 是否为大图路径。
- 检查 `data.js` 中该酒的 `largeImage` 是否为实际大图文件路径。

### 3. 底部菜单筛选不生效

**原因**：`types` 数组配置错误（如写成了字符串而非数组），或 `main.js` 中过滤逻辑有问题。

**排查**：
- 确保 `data.js` 中每款酒的 `types` 是数组（如 `['酱香']`），而不是字符串（`'酱香'`）。
- 在控制台执行 `console.log(wineData)` 查看数据结构是否正确。

### 4. 画廊图片切换卡顿或无法滑动

**原因**：触摸事件可能与其他手势冲突，或图片过大导致加载慢。

**优化**：
- 在 `main.js` 中调整 `touchstart` 和 `touchend` 的阈值（目前为 30px），可根据需要增大或减小。
- 压缩图片尺寸，减少加载时间。
